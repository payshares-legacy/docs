ledger_index

optional | default: current
A particular ledger to get this information from. It can either be a sequence number or one of the following:

  • current- The current view of paysharesd can change when the ledger is closed.
  • validated- The last validated ledger this paysharesd knows about. As long as this paysharesd is properly connected to the network, this ledger should only change as a result of a major catastrophe.

ledger_hash

optional
You can also specify a particular ledger by its hash to get this information from.
